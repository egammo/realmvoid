# REALMVOID

**Wer erinnert sich?**

REALMVOID is an open-source cyberpunk-noir shared universe franchise set in the sprawling megacity of Agnon City, 2226 AD. A world of crumbling infrastructure, outlawed technomancy, and memories that refuse to stay dead.

## What is REALMVOID?

REALMVOID is not a game engine. It is not a single product. It is **a world** — open for anyone to explore, play, and expand.

- **Franchise Bible** — The rules of the universe: four reality layers, technomancy, lineages, and the philosophy that holds it all together.
- **Casefile Player** — A browser-based text adventure engine. Play noir investigation cases set in Agnon City. No install required.
- **PnP Quickstart** — A tabletop one-shot you can play with friends, a handful of dice, and 30 minutes.
- **Community Casefiles** — Write your own cases using the M7 format. Contribute them back. Build the world together.

## The World

Agnon City exists across four layers of reality:

| Layer | What it is |
|---|---|
| **MEATSPACE** | The physical city. Rain, rust, neon, and concrete. |
| **NET** | The digital overlay. Data streams and ghost protocols. |
| **AETHER** | The space between. Forbidden technomancy and broken veils. |
| **DEAD ZONE** | Where nothing should exist. But something does. |

Every case crosses at least two layers. Every choice increases your **HEAT** — a measure of how close the system is to finding you. At HEAT 100, the system wins.
