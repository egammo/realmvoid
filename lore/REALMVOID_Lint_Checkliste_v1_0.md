# REALMVOID — Content Lint-Checkliste v1.0
## Qualitätsprüfung für Community-Module

**Zweck:** Jedes neue Modul (City Pack, Casefile, Technomancy-Effekt, Fraktion) durchläuft diese Checkliste vor Merge in die Canon Registry.

---

## 1. CORE-CANON-CHECK

- [ ] **Keine echte Magie.** Jeder Effekt hat eine technische Kausalkette.
- [ ] **Erdgebunden.** Keine Raumfahrt, keine Aliens, keine Paralleluniversen.
- [ ] **Ebenenmodell korrekt.** Nur MEATSPACE / NET / AETHER als Existenz-Ebenen.
- [ ] **Schlüssel korrekt.** Nur PACT / HELIX / OVERLAY (+ VOID-NOISE als Sonderfall).
- [ ] **HEAT vorhanden.** Jede Technomancy-Aktion hat benannte Kosten.
- [ ] **Aurora = Mythos.** Nie als bewiesene bewusste Entität dargestellt. Founding Myth, nicht Fakt.
- [ ] **VOID-NOISE = nicht kontrollierbar.** Kein Spieler/Charakter kann es auslösen oder steuern.

---

## 2. M4-REFERENZ-CHECK (Technomancy)

- [ ] **IDs existieren.** Jede referenzierte M4-ID (z.B. M4-HLX-07) existiert im aktuellen Katalog (v0.1.1).
- [ ] **Keine Reservebank-IDs.** Reservebank-Effekte haben KEINE M4-Nummern. Bei Nutzung: als narrativen Effekt beschreiben oder auf ähnlichen Core-Effekt verweisen.
- [ ] **Ebene ist strikt.** Nur MEATSPACE / NET / AETHER — kein „alle", kein „variabel".
- [ ] **HEAT ist Single-Type.** Ein Primärtyp + Stufe (niedrig/mittel/hoch/extrem). Sekundäre Kosten → Spuren.
- [ ] **Counterplay vorhanden.** Jeder Effekt muss brechbar sein.
- [ ] **Spuren vorhanden.** Jeder Effekt hinterlässt Nachweise. Keine spurlose Magie.
- [ ] **Ethik-Flags in Spuren.** „Ethisch fragwürdig" gehört nach Spuren (⚠), nicht in HEAT.

---

## 3. M7-CASEFILE-CHECK

- [ ] **Hook vorhanden.** Wer kommt? Wer zahlt? Wer lügt?
- [ ] **The Job ≠ The Dirt.** Offizieller Auftrag und wahre Lage sind unterschiedlich.
- [ ] **The Crossing vorhanden.** Mindestens 2 Existenz-Ebenen werden gekreuzt.
- [ ] **3 Schlüssel-Szenen.** Mindestens 1× Social/PACT, 1× NET/HELIX, 1× MEATSPACE/OVERLAY oder Dead Zone.
- [ ] **HEAT-Spike definiert.** Klarer Eskalationspunkt mit benanntem HEAT-Typ und Konsequenz.
- [ ] **Aftermath hat Varianten.** Mindestens 2 verschiedene Ausgänge mit unterschiedlichen Konsequenzen.
- [ ] **Truth Tag gesetzt.** Genau einer: `CORE-OK` · `SETTING-OK` · `FOUNDING-MYTH` · `LOCAL-TRUTH` · `ALT-TIMELINE` · `NONCANON`
- [ ] **Noir-Ton.** Klein anfangen, groß enden. Keine Weltenrettung, keine Heldenreise.

---

## 4. M1-CITY-PACK-CHECK

- [ ] **Zonen definiert.** Mindestens 3 Tiers mit je 3+ benannten Orten.
- [ ] **Dead Zone vorhanden.** Mindestens 1 Gebiet ohne Netz.
- [ ] **Pro Ort:** Name + 1–2 Sätze + Fraktionspräsenz.
- [ ] **Lokale Fraktionen.** Mindestens 2 stadtspezifische Fraktionen (nicht nur globale).
- [ ] **Fixer vorhanden.** Mindestens 1 Fixer pro aktive Fraktion mit: Name, Spitzname, Persönlichkeit, Treffpunkt, Job-Stil, Geheimnis.
- [ ] **Fixer-Abdeckung.** Jeder Fixer kommt in mindestens 1 Casefile vor (als Auftraggeber, Antagonist oder Kontakt).
- [ ] **NPC-Seeds.** Mindestens 10 Atmosphäre-NPCs mit: Name, Rolle, 1 Satz Persönlichkeit, 1 Satz Hook, Zone.
- [ ] **Ebenen-Spread.** Casefiles decken alle 3 Ebenen + Dead Zone ab.

---

## 5. M2-FRAKTIONS-CHECK

- [ ] **Ziel definiert.** Was will die Fraktion?
- [ ] **Kontrolle definiert.** Was besitzen/beherrschen sie?
- [ ] **Methode definiert.** Wie operieren sie?
- [ ] **Interner Konflikt.** Mindestens 1 Widerspruch innerhalb der Fraktion.
- [ ] **Noir-Regel.** Fraktion kann Auftraggeber UND Gegner sein.
- [ ] **Kein Schwarz/Weiß.** Keine rein böse oder rein gute Fraktion.

---

## 6. ALLGEMEINE KONSISTENZ

- [ ] **Keine neuen Ebenen.** Nur MEATSPACE / NET / AETHER.
- [ ] **Keine neuen Schlüssel.** Nur PACT / HELIX / OVERLAY (VOID-NOISE ist kein Schlüssel).
- [ ] **HEAT-Typen konsistent.** Nur: Prozessor-Hitze, Bandbreite, Audit-Spur, Hardware-Verschleiß, Körperbelastung, Brain-Burn, Zeit.
- [ ] **Truth Tags konsistent.** Nur: `CORE-OK` · `SETTING-OK` · `FOUNDING-MYTH` · `LOCAL-TRUTH` · `ALT-TIMELINE` · `NONCANON`
- [ ] **Versionsnummer angegeben.** Kompatibilitätsversion der Bible + M4 angeben.
- [ ] **Attribution vorhanden.** Mindestens: *„Basierend auf REALMVOID"*
- [ ] **Lizenz angegeben.** CC BY-SA 4.0 oder kompatibel.

---

## Quick-Pass (5-Sekunden-Check)

Vor dem Lesen eines neuen Moduls — diese 5 Fragen:

1. Gibt es echte Magie? → **Ablehnen.**
2. Verlässt es die Erde? → **Ablehnen.**
3. Ist Aurora eine bewiesene bewusste Person? → **Ablehnen.**
4. Gibt es Technomancy ohne HEAT? → **Zurück zum Autor.**
5. Gibt es einen Casefile ohne Crossing? → **Zurück zum Autor.**
